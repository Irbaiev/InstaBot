# InstaBot

Десктоп-приложение для автоматической отправки комментариев в Instagram с поддержкой нескольких аккаунтов и генерацией текста через локальную LLM (Ollama).

## Стек

- **Backend:** Python 3.11+, FastAPI, uvicorn, aiohttp
- **Frontend:** HTML/CSS/JS (встроен в pywebview-окно)
- **LLM:** Ollama (локально, без интернета)
- **Хранение:** `config.yaml` (аккаунты, посты, настройки)

Скачать и установить: https://ollama.com/download

```
ollama pull gemma3:4b
```

> Другие варианты: `llama3.2`, `mistral`, `llama3.2:1b` (лёгкая)

### 3. Зависимости Python

```bash

.venv\Scripts\activate
pip install -r requirements.txt
```

## Как пользоваться

### Шаг 1 — Получи файл куков

Куки нужны для авторизации без пароля. Используй расширение браузера [cookies.txt](https://chromewebstore.google.com/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc) для Chrome/Edge:

1. Войди в Instagram в браузере
2. Открой расширение на странице instagram.com
3. Нажми **Export** → сохрани файл в папку `cookies/`

Формат файла — Netscape (`.txt`). Пример названия: `cookies/account1.txt`

### Шаг 2 — Добавь аккаунт

В разделе **Аккаунты**:
- **Имя** — любое (для логов), например `main`
- **Файл куков** — путь к `.txt` файлу (или нажми 📂 для выбора)
- Нажми **Добавить**

### Шаг 3 — Добавь посты

В разделе **Посты** вставь ссылки на публикации Instagram:

```
https://www.instagram.com/p/ABC123/
https://www.instagram.com/reel/XYZ456/
```

Поддерживаются форматы: `/p/`, `/reel/`, `/tv/`

### Шаг 4 — Настрой параметры

| Параметр | Описание |
|----------|----------|
| Модель Ollama | Имя модели (должно совпадать с `ollama list`) |
| Режим | `all` — все аккаунты пишут все посты; `round_robin` — посты делятся между аккаунтами |
| Параллельность | Сколько аккаунтов работают одновременно |
| Задержка мин/макс | Пауза между комментариями (секунды) |
| Повторов | Сколько раз повторять при ошибке |

Нажми **💾 Сохранить** чтобы настройки сохранились в `config.yaml`.

### Шаг 5 — Запусти

Нажми **▶ Запустить**. Логи появятся в реальном времени.

## Генерация комментариев

Бот генерирует комментарии двумя способами:

1. **Ollama (приоритет)** — локальная LLM, контекстуальный комментарий по подписи к посту
2. **Fallback** — шаблонные фразы по категориям (beauty, food, nature, gaming, sport, selfie) если Ollama недоступна

Ollama должна быть запущена перед стартом бота. Бот автоматически ждёт загрузки модели в память перед началом работы.

## Troubleshooting

**Ошибка "Сервер не ответил за 15 секунд"**
- Порт 8765 занят другим процессом. Смени порт в `backend/config.py`

**"Куки просрочены"**
- Сессия Instagram истекла. Экспортируй куки заново.

**Комментарии из fallback вместо Ollama**
- Убедись что Ollama запущена: в трее должна быть иконка, или выполни `ollama serve`
- Проверь имя модели: `ollama list` — скопируй NAME точно

**503 при запросе к Ollama**
- Модель не загружена в память. Бот ждёт загрузки автоматически (до 2 минут).
- Если не помогает — запусти `ollama run <model> "test"` вручную, потом снова запусти бота.
